passwd-sso ブラウザ拡張機能

最終更新日: 2026-03-11

概要

passwd-sso ブラウザ拡張機能（以下「本拡張機能」）は、passwd-sso パスワードマネージャーの補助ツールです。本ポリシーでは、本拡張機能がアクセスするデータ、その使用方法、およびお客様の権利について説明します。

データの収集

本拡張機能は、個人データを第三者に収集・送信・販売することはありません。すべての保管庫データはエンドツーエンドで暗号化され、お客様のブラウザ内でのみ復号されます。

ローカルでアクセスするデータ

本拡張機能は、お客様のデバイス上でのみ以下のデータにアクセスします：

  • storage — 拡張機能の設定および暗号化されたセッション状態（chrome.storage 経由）
  • activeTab — 保存された認証情報とのマッチングおよびオートフィルのための現在のタブの URL
  • clipboard — お客様の操作に応じて、パスワードまたはユーザー名を一時的にクリップボードに書き込み
  • forms — オートフィル機能を提供するための Web ページ上のログインフォームフィールド

サーバーとの通信

本拡張機能は、お客様がセルフホストする passwd-sso サーバーインスタンスとのみ通信します。他のサーバーにデータが送信されることはありません。すべての通信は HTTPS および暗号化されたペイロードを使用します。

ブラウザの権限

本拡張機能は以下の権限を要求します：

  • storage — 拡張機能の設定および暗号化されたセッション状態の保存
  • alarms — 設定可能なタイムアウト期間後に保管庫を自動ロック
  • activeTab — 認証情報のマッチングとオートフィルのために現在のページ URL を検出
  • scripting — Web ページのフォームにオートフィルスクリプトを注入
  • contextMenus — 右クリックメニューからクイックコピーおよびオートフィルを提供
  • clipboardWrite — 要求に応じてパスワードとユーザー名をクリップボードにコピー
  • offscreen — バックグラウンドでクリップボード操作を実行
  • hostPermissions — フォーム検出とオートフィルのために Web ページにアクセス（要求時に付与）

セキュリティ

すべてのパスワードデータは、送信前にクライアント側で AES-256-GCM を使用して暗号化されます。マスターパスフレーズがデバイスから離れることはありません。鍵導出には 600,000 回の反復を持つ PBKDF2 を使用しています。

本ポリシーの変更

本ポリシーは随時更新される場合があります。変更は更新日とともに本ページに反映されます。
